# 데이터 분석 및 시각화 대시보드

CSV 데이터를 업로드하면 **데이터 품질 분석**, **시각화**, **머신러닝 분석**을 수행하고 대시보드로 결과를 보여주는 웹 애플리케이션입니다.

## 🚀 주요 기능

### 1. 📊 데이터 개요
- 기본 통계 정보 (행, 열, 결측치, 중복)
- 데이터 샘플 미리보기
- 컬럼 정보 및 데이터 타입 확인

### 2. 🔍 데이터 품질 분석
- **결측치 분석**: 각 컬럼의 결측치 현황 및 비율
- **중복 분석**: 중복 행 감지 및 통계
- **데이터 타입 검증**: 컬럼별 데이터 타입 확인
- **이상치 탐지**: IQR 기반 이상치 감지
- **범위 분석**: 수치형 변수의 최소/최대/평균/표준편차

### 3. 📈 데이터 시각화
- **상관계수 히트맵**: 수치형 변수 간 상관관계 시각화
- **분포도**: 히스토그램 및 빈도 분포
- **상자 그림**: 이상치 및 범위 시각화
- **원형 그래프**: 카테고리형 데이터 비율
- **산점도**: 두 변수 간 관계 분석
- **다중 변수 분석**: 여러 변수 간 패턴 탐색

### 4. 🤖 머신러닝 분석
- **K-Means 클러스터링**: 데이터를 K개 그룹으로 분류
- **특성 중요도**: 타겟 변수 예측에 중요한 특성 분석
- **엘보우 곡선**: 최적 클러스터 수 결정 지원

### 5. 📉 고급 분석
- **요약 통계**: 상세한 통계 정보 테이블
- **데이터 필터링**: 조건에 따른 데이터 필터링 및 다운로드
- **분석 리포트**: 전체 분석 결과 리포트 생성

## 📋 설치 및 실행

### 필수 요구사항
- Python 3.8 이상

### 1단계: 의존성 설치
```bash
pip install -r requirements.txt
```

### 2단계: 애플리케이션 실행
```bash
streamlit run app.py
```

### 3단계: 브라우저에서 접속
자동으로 브라우저가 열리거나 http://localhost:8501로 접속합니다.

## 📊 사용 방법

### 샘플 데이터 템플릿
1. 사이드바의 **"📥 샘플 CSV 다운로드"** 버튼 클릭
2. 샘플 파일 다운로드 및 분석

### 직접 데이터 업로드
1. 샘플 파일과 같은 형식으로 CSV 파일 준비
2. 사이드바의 파일 업로더에서 파일 선택
3. 각 탭에서 다양한 분석 결과 확인

### 데이터 템플릿 구조
```
customer_id     : 고객 ID (정수)
name           : 고객 이름 (텍스트)
age            : 연령 (정수)
gender         : 성별 - M 또는 F (텍스트)
region         : 지역 - North/South/East/West (텍스트)
product_category: 제품 분류 (텍스트)
purchase_amount : 구매 금액 (실수)
purchase_date   : 구매 날짜 - YYYY-MM-DD 형식 (날짜)
satisfaction_score: 만족도 점수 1-5 (정수)
loyalty_member  : 충성도 멤버 여부 - Yes 또는 No (텍스트)
```

## 💡 팁

1. **데이터 포맷**: CSV 파일은 UTF-8 인코딩 사용 권장
2. **날짜 형식**: YYYY-MM-DD 형식 사용 (2024-01-15)
3. **수치 데이터**: 쉼표 없이 소수점(.)으로 표기
4. **카테고리 데이터**: 공백 제거 후 일관성 있게 입력

## 🐛 문제 해결

### 파일 업로드 불가
- CSV 파일이 UTF-8로 인코딩되었는지 확인
- 파일 크기가 200MB 이하인지 확인

### 분석 오류
- 데이터에 필수 컬럼이 모두 있는지 확인
- 데이터 타입이 올바른지 확인 (날짜, 숫자 등)
- 결측치가 너무 많지는 않은지 확인

## 📞 지원

문제가 발생하면:
1. 샘플 파일로 테스트
2. 데이터 형식 확인
3. requirements.txt 라이브러리 재설치
